#### 대화 주제: T-Flow AI 영상 자동화 툴 분석

**참여자**: 
- 기획/개발 담당자

**논의 내용**:

### 1. 프로젝트 개요 페이지 분석
- **기능**: 새 프로젝트 생성, 프로젝트 개요 확인
- **구성 요소**:
  - 프로젝트 정보 (제목, 설명)
  - 작업 진행 상태
  - 생성된 콘텐츠 목록
  - 프로젝트 설정

### 2. 대본 업로드 시스템 분석
- **업로드 방식**:
  - AI 생성 (인공지능이 대본 자동 생성)
  - 파일 업로드 (기존 문서 업로드)
  - 비디오 업로드 (영상에서 대본 추출)
  - 직접 입력 (수동 입력)

- **AI 생성 설정**:
  - 언어 선택 (한국어, 영어, 일본어, 중국어 등)
  - 장르 선택 (스릴/미스터리, SF/판타지, 로맨스, 코미디, 액션, 공포, 드라마, 다큐멘터리, 역사, 전쟁, 뮤지컬, 스포츠, 범죄, 정치, 가족, 성장, 교육, 동물, 음식, 여행)
  - 톤/스타일 선택 (유머러스, 진지, 감성적, 긴장감, 따뜻함, 어두움, 밝음, 차분함, 활기참, 신비로움)
  - 등장인물 수 선택 (3명, 4명, 5명)
  - 스토리 길이 선택 (짧음, 보통, 김)
  - Human Touch 설정 (감정, 교육적 목표, 정보 제공 등)
  - 창작 난이도 (Strict - 엄격한 규칙, Balanced - 균형, Creative - 창의적)

### 3. 대본 생성 및 편집
- **대본 생성 과정**:
  1. 제목 및 줄거리 선택 (AI가 생성한 10개 후보 중 선택)
  2. 줄거리 선택 확인 (스토리 구조, 주제, 톤 등 확인)
  3. 대본 생성 (대사 비율 설정 가능 - 예: 60%)
  4. 최종 대본 확인 및 편집

### 4. 캐릭별 TTS (Text-to-Speech) 설정
- **TTS 엔진 선택**:
  - ElevenLabs (고품질, 유료)
  - Microsoft Azure (묶음형, 묶음형 Pro)
  - Google TTS
  - OpenAI TTS

- **캐릭터 설정**:
  - 성별 선택 (남자/여자)
  - 목소리 선택 (다양한 음색 제공)
  - 속도, 피치, 볼륨 조절
  - 캐릭터별 음성 미리듣기

- **TTS 생성 완료 후**:
  - 전체 오디오 병합 기능
  - 개별 음성 파일 다운로드
  - 음성 길이 확인

### 5. 이미지 & 영상 관리 (Scene Generator)
- **스타일 설정**:
  - Art Style 선택 (Anime, Realistic, 3D, Pixel Art 등)
  - 캐릭터 일관성 설정 (Consistent Character)
  - 이미지 품질 설정

- **장면 생성**:
  - AI 이미지 생성
  - 업로드된 이미지 활용
  - 장면 갤러리 (Scene Gallery)에서 템플릿 선택
  - 텍스트가 포함된 이미지 생성 가능

- **씬 편집**:
  - 씬 추가/삭제/편집
  - 씬 순서 변경
  - AI 자동 씬 매칭

### 6. 이미지-자막 동기화

### 9. 영상 생성 및 납품

## 분석된 주요 기능 요약

### 핵심 기능
1. **AI 대본 생성**: 장르, 톤, 등장인물 설정 후 AI가 자동으로 대본 작성
2. **TTS 음성 생성**: 다양한 AI 음성 엔진을 활용한 자연스러운 음성 합성
3. **AI 이미지 생성**: 콘텐츠 스타일에 맞는 장면 이미지 자동 생성
4. **자동 동기화**: 음성 길이에 맞춘 이미지-자막 자동 동기화
5. **타임라인 편집**: 전문적인 영상 편집 기능 제공
6. **자막 스타일링**: 다양한 자막 템플릿과 커스터마이징
7. **멀티 플랫폼 납품**: 브루, 캡컷 등 다양한 편집 툴로 납품 가능

### UI/UX 특징
- 직관적인 단계별 프로세스 (대본 업로드 → TTS 설정 → 이미지 생성 → 동기화 → 편집 → 납품)
- 다크 테마 기반의 현대적인 디자인
- 실시간 미리보기 기능
- 진행 상태 시각화
